## ssb-graphql-defaults

GraphQL schema for Secure Scuttlebot, meant to be used with [open-app-graphql-server](https://github.com/open-app/open-app-graphql-server).

## Usage

`npm i -S ssb-graphql-defaults`

## API

### Query
- whoami
- history
- profile
- channels

### Mutation
- postMessage
- aboutMessage

### Subscription
- gossip
- progress
